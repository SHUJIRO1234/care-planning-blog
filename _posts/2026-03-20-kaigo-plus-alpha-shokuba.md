## 「介護の仕事＝介護だけ」だと思っていませんか？

「介護の仕事に興味はあるけど、毎日同じことの繰り返しでは…」
「前職のスキルが活かせないのでは…」

そんな心配は無用です。実は、**介護の仕事にプラスαの役割**がある職場が増えています。

この記事では、介護だけでなく農業・eスポーツ・研修講師など、**ユニークな＋αがある職場**をご紹介します。

## ケア・プランニングの「介護＋α」モデル

東京都荒川区のケア・プランニング株式会社では、介護業務に加えて**プラスαの役割**を持つ働き方を推進しています。

| 基本業務 | ＋αの役割 | こんな人に向いている |
|---------|----------|-------------------|
| 介護職 | **さんさんふぁーむ運営** | 農業・園芸が好きな方 |
| デイサービス | **eスポーツリハビリ** | ゲーム・IT好きな方 |
| 訪問介護 | **初任者研修の講師補助** | 教えることが好きな方 |
| 看護師 | **AI記録の社内トレーナー** | 新しいことに挑戦したい方 |
| PT/OT | **eスポーツリハビリ開発** | 企画・開発に興味がある方 |
| 外国人材 | **母国語での研修サポート** | 語学力を活かしたい方 |

## 介護×農業：さんさんふぁーむ

### 水耕栽培レタスを育てる介護の仕事

ケア・プランニングが運営する**「さんさんふぁーむ」**は、東京都ソーシャルファーム認可を受けた水耕栽培施設です。

| 項目 | 内容 |
|------|------|
| 栽培品目 | レタス（水耕栽培） |
| 認可 | 東京都ソーシャルファーム |
| 活用先 | デイサービスの食事・配食サービス |

介護職員がさんさんふぁーむの運営にも関わることで、**「育てる→食べてもらう→喜んでもらう」**というサイクルを体験できます。利用者様と一緒に栽培する活動は、リハビリや生きがい支援としても効果的です。

**こんな方に向いています：**
- 植物を育てるのが好き
- 食に興味がある
- 屋内作業で体力的に無理がない仕事も持ちたい

## 介護×eスポーツ：eほっと

### ゲームでリハビリする新しいデイサービス

**eスポーツデイサービス「eほっと」**では、ゲームを活用した認知機能トレーニングや手指のリハビリを提供しています。

| eスポーツの介護効果 |
|-------------------|
| 手指の巧緻性向上 |
| 認知機能の維持・改善 |
| 社会的交流の促進 |
| モチベーション維持 |

スタッフは利用者様と**一緒にゲームを楽しみながら**リハビリをサポートします。「ゲームが好き」「ITに興味がある」方にとって、これほど楽しい介護の仕事はないでしょう。

**こんな方に向いています：**
- ゲームやeスポーツに興味がある
- デジタル機器の操作が得意
- 従来のレクリエーションに物足りなさを感じる

## 介護×教育：初任者研修講師

### 自社スクールで教える介護職

ケア・プランニングは**介護職員初任者研修スクール**を自社で運営しています。経験を積んだ介護職員が、講師補助として研修に関わることができます。

| 役割 | 内容 |
|------|------|
| 講師補助 | 実技指導のサポート・受講者のフォロー |
| 体験談共有 | 現場のリアルなエピソードを伝える |
| 外国人コース | GCC連携の外国人特化コースのサポート |

「教えることが好き」「後輩を育てたい」という方に、介護の仕事＋教育という**ダブルのやりがい**があります。

## 介護×テクノロジー：AI社内トレーナー

### AIツールを現場に広める役割

ケア・プランニングでは、介護記録AIや業務分析アプリなど、複数のテクノロジーを導入しています。これらを現場に定着させるための**社内トレーナー**の役割があります。

看護師やベテラン介護職が、新人スタッフに**「AIツールの使い方」を教える**ポジションです。

**こんな方に向いています：**
- 新しいツールやアプリに抵抗がない
- 人に教えるのが好き
- 「現場を良くしたい」という改善意欲がある

## なぜ「介護＋α」が可能なのか？

ケア・プランニングが「介護＋α」を実現できる理由は、**複数のサービスを一つの会社で運営**しているからです。

| サービス | ＋αの可能性 |
|---------|-----------|
| 訪問介護（ヘルパー34名） | 研修講師・AI指導 |
| 訪問看護 | データ分析・多職種連携 |
| デイサービス | eスポーツ・さんさんふぁーむ |
| リハビリ型訪問看護（NEW OPEN） | リハビリプログラム開発 |
| 初任者研修スクール | 教育・外国人サポート |
| 福祉用具 | 環境調整・提案営業 |

小規模な事業所では「介護だけ」になりがちですが、**複合的にサービスを展開しているからこそ、多彩な＋α**が可能です。

## よくある質問

**Q. ＋αの仕事は強制ですか？**

A. いいえ。「やりたい」と思う方にチャンスを提供するスタイルです。まずは介護の仕事を覚えて、興味があれば＋αに挑戦できます。

**Q. ＋αの分、給料は上がりますか？**

A. 役割に応じた手当やキャリアアップが見込めます。特にAIトレーナーや研修講師はスキルとして評価されます。

**Q. 介護未経験でも＋αに挑戦できますか？**

A. まずは介護の基礎を身につけてから（3ヶ月〜半年が目安）。その後、興味のある分野に挑戦できます。自社スクールで初任者研修を取得できるので、未経験からのスタートも安心です。

## まとめ

介護の仕事は、もはや**「介護だけ」ではありません**。

- 農業（さんさんふぁーむ）
- eスポーツ（eほっと）
- 教育（初任者研修講師）
- テクノロジー（AI社内トレーナー）
- 国際交流（外国人材サポート）

あなたの「好き」や「得意」を活かせる介護の職場が、ここにあります。
