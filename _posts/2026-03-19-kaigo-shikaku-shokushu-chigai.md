## 介護の仕事にはどんな種類がある？自分に合った働き方を見つけよう

「介護の仕事に興味はあるけど、種類が多くてどれが自分に合うかわからない…」

この記事では、介護業界の主な職種を**わかりやすく比較**して、あなたに合った働き方を見つけるお手伝いをします。

## 職種別の詳しい比較

### 訪問介護ヘルパー

利用者様のご自宅を訪問し、**身体介護（入浴・排泄・食事）や生活援助（掃除・調理・買い物）**を提供します。

| 項目 | 内容 |
|------|------|
| 1日の訪問件数 | 4〜6件 |
| 魅力 | 一人ひとりと深い信頼関係を築ける |
| 大変なこと | 天候に左右される・一人での判断 |
| 向いている人 | 自立して動ける方・コミュ力のある方 |
| 未経験 | **OK**（初任者研修は入社後取得可） |

### デイサービス介護職

利用者様が日帰りで通う施設で、**レクリエーション・入浴介助・食事介助・リハビリ補助**を行います。

| 項目 | 内容 |
|------|------|
| 勤務時間 | 8:30〜17:30が一般的 |
| 魅力 | チーム制・夜勤なし・レクが楽しい |
| 大変なこと | 複数の利用者様を同時に見る |
| 向いている人 | チームワーク重視・明るい方 |
| 未経験 | **OK**（資格なしでも可） |

### 訪問看護師

主治医の指示のもと、利用者様のご自宅で**バイタルチェック・服薬管理・褥瘡ケア・リハビリ指導**などの看護ケアを提供します。

| 項目 | 内容 |
|------|------|
| 1日の訪問件数 | 4〜6件 |
| 魅力 | 看護の総合力が身につく |
| 大変なこと | 一人での臨床判断・オンコール |
| 向いている人 | じっくり患者と関わりたい看護師 |
| ブランク | **OK**（同行訪問研修あり） |

### PT・OT（訪問・デイサービス）

**訪問看護ステーション**または**デイサービス**で、利用者様のリハビリテーションを担当します。

| 項目 | 訪問PT・OT | デイサービスPT・OT |
|------|-----------|-----------------|
| 場所 | 利用者様の自宅 | デイサービス施設 |
| 魅力 | 生活に直結するリハビリ | eスポーツ等の新しいアプローチ |
| 対象 | 在宅の利用者様 | 通所の利用者様 |
| 勤務 | 日勤 | 日勤のみ |

## あなたに合った職種の選び方フローチャート

**Q1. 看護師・PT・OTの資格を持っていますか？**

→ **はい** → Q2へ
→ **いいえ** → Q3へ

**Q2. 一人で訪問する働き方と、施設でチームと働く方、どちらが好みですか？**

→ **一人で訪問** → 訪問看護 or 訪問PT・OT
→ **チームで働く** → デイサービスのPT・OT or 看護師

**Q3. 一人で動くのが好きですか？チームが好きですか？**

→ **一人で動く** → 訪問介護ヘルパー
→ **チームが好き** → デイサービス介護職
